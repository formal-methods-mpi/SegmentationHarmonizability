# Segmentation Harmonizability
This is the repository for the segmentation harmonizability study. The harmonizability of volumetrics derived with different scanners and/or segmentations will be evaulated.

# Overview
|Question|Answer  |
| :---   |  :---- |
| What is the project about?| Evaluating the performance of harmonisation tools for brain volumetrics under conditions typical for integrative neuroimaging analyses. |
| How would the project look like if completed? | An accepted paper. Foudations for decision-making in other reserach projects.  |
|How does the project benefit the group? |  Prerequisite for ongoing research projects. Application of ICED approach.  |
| How does the project impact the wider scientific comunity? | Inform decision making in neuroiamging IDAs. Might save resources by providing alternatives to reprocessing. |
| What do you expect us to invest to get the project over the finish line? | Andreas will assisst Laurenz in the methodological elaboration of the project. Laurenz will conduct the analyses, illustrate the results and write the eventual manuscript |
| Timeline | Preprint-ready at the end of February |
